Tables Attributes

HTML5 tables attribute
Attribute Elements Purpose Example
colspan <th>, <td> Number of columns that the cell is to span
  • <th colspan="4">
  • <td colspan="3">
rowspan <th>, <td> Number of rows that the cell is to span
  • <th rowspan="3">
  • <td rowspan="2">
scope <th> Specifies which cells the header cell applies to
  • <th scope="col">
  • <th scope="row">
  • <th scope="rowgroup">
  • <th scope="colgroup">
id Commonly use on <th> Specifies element unique identifier
  • <th id="h">
headers <th>, <td> The headers for this cell
  • <th id="h1" headers="h">
  • <td headers="h h1">
abbr <th> Alternative label to use for the header cell when referencing the cell in other contexts
  • <th abbr="Make">Toy manufacturer<th>
  • <th abbr="Model">Vehicle model<th>
border <table> Explicitly indicate the table is not being used for layout.
  • <table border="">
  • <table border="1">

Additional attribute: sortable on <table> elements means to enables a sorting interface for the table. And sorted on <th> elements means column sort direction and ordinality.
